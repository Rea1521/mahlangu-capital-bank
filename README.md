# Mahlangu Capital Bank - Banking Management System

A secure, cloud-hosted banking management system with a modern black and gold UI, inspired by Capitec. Built with React (frontend), Spring Boot (backend), and PostgreSQL (database).

![Mahlangu Capital Bank Logo](frontend/src/assets/logo.png)

### Core Features
- **Customer Management**: Register and login with secure password hashing
- **Account Management**: Create Savings (black card) or Credit (gold card) accounts
- **Transactions**: Deposit, withdraw, and transfer funds
- **Card Management**: View card details, freeze/unfreeze cards, manage daily limits

### Banking Features
- **Card Details**: View card number (masked with show/hide option), expiry, and status
- **Daily Limits**: Track and manage daily transaction limits
- **Transaction History**: View recent transactions with categories
- **Account Settings**: Manage profile, security settings, and account status
- **PDF Statements**: Generate and download monthly statements
- **Rewards Program**: Cashback and special offers

### UI/UX
- **Black & Gold Theme**: Premium banking aesthetic
- **Card Design**: 
  - Savings accounts: Black cards with gold borders
  - Credit accounts: Gold gradient cards
- **Bottom Navigation**: Quick access to Home, Cards, Transact, Messages, Explore
- **Responsive Design**: Works on desktop and mobile
